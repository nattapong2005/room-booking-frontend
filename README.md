# ระบบจองห้องประชุม วิทยาลัยอาชีวศึกษานครปฐม

ระบบจองห้องประชุม Frontend พัฒนาด้วย React, TypeScript, และ TailwindCSS

## ฟีเจอร์
- แสดงรายการห้องประชุม 4 ห้อง พร้อมสถานะ (ว่าง/ไม่ว่าง)
- ตารางปฏิทินการจองรายสัปดาห์
- ระบบจองห้องประชุม (จำลองข้อมูล)
- รองรับการแสดงผลภาษาไทย

## การติดตั้งและใช้งาน

1. ติดตั้ง Dependencies:
   ```bash
   npm install
   ```

2. รันโหมด Development:
   ```bash
   npm run dev
   ```

3. สร้าง Production Build:
   ```bash
   npm run build
   ```

## เทคโนโลยี
- Vite
- React 19
- TypeScript
- TailwindCSS v4
- Lucide React (Icons)
- Date-fns (จัดการวันที่)
